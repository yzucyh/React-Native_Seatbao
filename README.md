# React-Native_Seatbao
2019元智大學創意APP佳作
>主要技術 : React-Native + Nodejs + firebase<br>
# 功能介紹
1. 手機端預約圖書館座位，資料送至firebase完成預約<br>
2. 掃描圖書館位置QRCode，解碼後與token比較是否一致完成報到<br>
3. 串聯校園帳號，不須額外註冊<br>
4. 休息倒數機制，以中午休息為例<br>
